遊戲說明 佛地魔正在崛起，必須將它殺掉。玩家可自行設置佛地魔的 HP, Power, Knowledge，並可輸入10名成員且設置姓名、初始等級與角色，用以對抗佛地魔。每個角色的屬性不同，可自行搭配。

如何執行遊戲:編譯 遊戲.cpp 然後執行

輸入:(佛地魔的) HP>> Power >> Knowledge >> 成員姓名 >> 成員初始等級 >>成員角色 (可輸入10名  輸入完按Ctrl+z Enter)
